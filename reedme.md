# BabiPoly Website

A modern, responsive website project for **Beirut Poly** — an engineering and technology faculty based in Beirut.  
The project is built with **Bootstrap 5**, HTML5, CSS3, and JavaScript to deliver a clean and accessible user experience.

---

## 🚀 Features
- **Responsive Design**: Mobile-first layout using Bootstrap grid system.  
- **Multi-section Homepage**: Includes Hero, About, Programs, Labs, Admissions, and Contact sections.  
- **Interactive Elements**: Smooth scroll, back-to-top button, and simple client-side form handling.  
- **Bootstrap Components**: Navbar, cards, modals, icons, forms, and more.  
- **Custom Styling**: Additional CSS for branding and polished design.
